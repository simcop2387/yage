class Model : yage.resource.geometry.Geometry;
A Model is a 3D object, often loaded from a file.

Each model is divided into one or more Meshes; each Mesh has its own material and an array of triangle indices that correspond to vertices in the Model 's vertex array. ModelNodes can be used to create 3D models in a scene.

Get an array of all of the Model's Joints, which are used for skeletal animation. This can be traversed as an array, or as a tree since each Joint references its parent and children.
