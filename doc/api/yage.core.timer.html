class Timer ;
A cross-platform, high-performance (microsecond) timing class.

Example:
 Timer a = new Timer();
 real b = a.get();		// b stores the current time.


this();
Initialize and start the Timer.

this(Timer rhs);
Copy Constructor

Params:
Timer rhs This Timer will be a copy of rhs.

real get ();
Return the Timer's time in seconds

bool getPaused ();
Is the Timer paused?

void setPaused (bool paused);
Set whether the Timer is paused.

void pause ();
Alias of setPaused(true )

void resume ();
Alias of setPaused(false )

void reset ();
Reset the Timer to zero.

void set (double time);
Set the Timer

Params:
double time Measured in seconds.
