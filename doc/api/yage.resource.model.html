class Model ;
A Model is a 3D object, typically loaded from a file. A model contains an array of vertices, texture coordinates, and normals. Each model is divided into one or more Meshes; each Mesh has its own material and an array of triangle indices that correspond to vertices in the Model 's vertex array. ModelNodes can be used to create 3D models in a scene.

void loadMs3d (char[] filename);
Load a model from a Milkshape3D model file. All materials, etc. referenced by this model are loaded through the Resource manager to avoid duplicates. Meshes without a material are assigned a default material. Uploading vertex data to video memory must be done manually by calling Upload(). This function needs to be tested on big-endian systems.
