class Device ;
The device class exists to group functions for initializing a window, checking OpenGL extensions, and other utility and lower level tasks.

static void init (int width, int height, ubyte depth, bool fullscreen, ubyte samples = cast(ubyte)32);
This function creates a window with the specified width and height in pixels. It also initializes an OpenAL context so that audio playback can occur. It must be called before most other code.

Params:
int width Width of the window in pixels
int height Height of the window in pixels
ubyte depth Color depth of each pixel (should be 16, 24 or 32)
bool fullscreen The window is fullscreen if true ; windowed otherwise.
ubyte samples The level of anti-aliasing.

static bool checkExtension (char[] name);
Searches to see if the given extension is supported in hardware.

static char[][] getExtensions ();
Return an array of all supported OpenGL extensions.

static int getLimit (int constant);
Get the number from the hardware specified by constant.

Params:

static bool getSupport (int constant);
Get whether the given opengl feature is supported by the hardware.

Params:

static float getAspectRatio ();
Return the aspect ratio (width/height) of the rendering window.

static uint getWidth ();
Return the current width of the window in pixels.

static uint getHeight ();
return the current height of the window in pixels.

static void resizeViewport (int _width, int _height, float near, float far, float fov, float aspect);
Resize the viewport to the given size. Special values of zero scale the viewport to the window size.

static void resizeWindow (int width, int height);
Stores the dimensions of the current window size. This is called by a resize event in Input.checkInput().
