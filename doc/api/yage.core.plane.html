struct Plane ;
A class representing a plane in 3D space. This is defined as a struct instead of a class so it can be created and destroyed without any dynamic memory allocation.

static Plane opCall ();
Return a Plane with all values at zero.

static Plane opCall (Vec3f n, float d);
Return a Plane with the Vec3f component of n and distance of d.

static Plane opCall (float x, float y, float z, float d);
Return a Plane with a Vec3f component of x, y, z and distance of d.

float getDistance (Vec3f point);
Get the distance from the center of this plane to the given point. This is useful for determining which side of the plane the point is on.

Plane normalize ();
Normalize the plane. This should be tested for accuracy.
