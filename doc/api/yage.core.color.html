struct Color ;
A struct used to represent a color. Colors are represented in RGBA format. Note that uints and dwords store the bytes in reverse, so Color (0x6633ff00).hex == "00FF3366" All Colors default to transparent black.

static Color opCall (char[] string);
Convert a string to a color. The string can be a 6 or 8 digit hexadecimal or an English color name. Black, blue, brown, cyan, gold, gray/grey, green, indigo, magenta, orange, pink, purple, red, violet, white, and yellow are supported.
