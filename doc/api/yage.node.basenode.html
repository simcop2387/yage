class BaseNode ;
Node and Scene both inherit from BaseNode and this abastract class defines fields and methods that are shared between them. Having both derrived from the same class also allows functions such as setParent( BaseNode parent) to take an argument of either type.

this();
Construct.

BaseNode addChild (Node child);
Add a child Node. Automatically detaches it from any other nodes.

Returns:
A self reference.

Node[] getChildren ();
Get an array of this Node's children

int getIndex ();
Get the index of this Node in its parent's array. Returns -1 if this node is a Scene.

float getLifetime ();
Get the time before the Node will be removed.

Return this Node's parent.

Scene getScene ();
Get the Scene at the top of the tree that this node belongs to.

char[] getType ();
Get the type of this Node as a string; i.e. "yage.node.node.ModelNode".

void onUpdate (void delegate(BaseNode self) on_update);
Set a function that will be called every time this Node is updated.

Params:
void delegate(BaseNode self) on_update the function that will be called. Use null as an argument to clear the function.

BUGS:
Certain Node methods cause access violations. Perhaps this is a dmd bug?

Example:
 SpriteNode s = new SpriteNode(scene);
 s.setMaterial("something.xml");

 // Gradually fade to transparent as lifetime decreases.
 void recolor(BaseNode self)
 {   (cast(SpriteNode)self).setColor(1, 1, 1, self.getLifetime()/5);
 }
 s.setLifetime(5);
 s.onUpdate(&doSomething);


void setLifetime (float seconds);
The Node will be removed (along with all of its children) after a given time.

Params:
float seconds The number of seconds before the Node will be removed. Set to float.infinity to make the Node last forever (the default behavior).

char[] toString ();
Return a string representation of this Node for human reading.

char[] toString (bool recurse);
Return a string representation of this Node for human reading.

Params:
bool recurse Print this Node's children as well.

void update (float delta);
Update the positions and rotations of this Node and all children by delta seconds.

protected synchronized void calcTransform ();
Calculate and store the absolute transformation matrices of this Node up to the first node that has a correct absolute transformation matrix. This is called automatically when the absolute transformation matrix of a node is needed. Remember that rotating a Node's parent will change the Node's velocity.
