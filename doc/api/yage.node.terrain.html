class TerrainNode : yage.node.node.Node;
A TerrainNode generates a landscape for rendering from a heightmap image. This class may change in the future as featurs are added.

Example:
 TerrainNode a = new TerrainNode(scene);  // Child of scene
 a.setScale(1000, 100, 1000);             // Make it a decent size
 a.setMaterial("terrain/islands.xml");
 a.setHeightMap("terrain/islands-height.png");

Construct this TerrainNode as a copy of another TerrainNode and recursively copy all children.

Params:
BaseNode parent This TerrainNode will be a child of parent.
TerrainNode original This TerrainNode will be an exact copy of original.

void setHeightMap (Image grayscale, float repeat = cast(float)1);
void setHeightMap (char[] image_file, float repeat = 1F);
Generate the 3D landscape rendered for this Node, from a heighmap image. The terrain generated always fits inside a 1x1x1 cube. Use setScale() to adjust to a comfortable size.

Params:
Image grayscale An image to load. It will be converted to a grayscale image if necessary. Whiter regions will be higher altitudes.
float repeat The material texture coordinates will be repeated this many times.
