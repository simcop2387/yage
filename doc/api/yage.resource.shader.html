class Shader ;
A Shader is a class used to represent a vertex or fragment shader. Material layers link shader objects together to form shader programs. When a node that uses that material is rendered, the shader program is applied.

this(char[] filename, bool type);
Construct this shader and then load and compile the given shader file.

Params:
char[] filename The shader source code file to load.
bool type Set to 0 for a vertex shader or 1 for a fragment shader.

char[] getSource ();
Return the source filename of this shader.

char[] getCode ();
Return the source code of this shader.

uint getShader ();
Return the OpenGL handle of the compiled shader.

char[] getCompileLog ();
Get messages from the shader compiler.
