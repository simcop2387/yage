yage.system.constant

Authors:
Eric Poggel

License:
LGPL

Constants used as parameters to various functions throughout the engine.

const int DEVICE_FBO ;
Hardware support for rendering directly to a texture (Frame Buffer Object)

const int DEVICE_MULTITEXTURE ;
Hardware support for using multiple textures in a single rendering pass

const int DEVICE_NON_2_TEXTURE ;
Hardware support for textures of arbitrary size

const int DEVICE_SHADER ;
Hardware support for openGl vertex and fragment shaders

const int DEVICE_VBO ;
Hardware support for caching vertex data in video memory (Vertex Buffer Object)

const int DEVICE_MAX_LIGHTS ;
Maximum number of lights that can be used at one time

const int DEVICE_MAX_TEXTURE_SIZE ;
Maximum allowed size for a texture

const int DEVICE_MAX_TEXTURES ;
Maximum number of textures that can be used in multitexturing

const int LIGHT_DIRECTIONAL ;
A light that shines in one direction through the entire scene

const int LIGHT_POINT ;
A light that shines outward in all directions

const int LIGHT_SPOT ;
A light that emits light outward from a point in a single direction

const int TEXTURE_FILTER_DEFAULT ;


const int TEXTURE_FILTER_NONE ;


const int TEXTURE_FILTER_BILINEAR ;


const int TEXTURE_FILTER_TRILINEAR ;


const int IMAGE_FORMAT_GRAYSCALE ;
A grayscale image

const int IMAGE_FORMAT_RGB ;
An image with red, green, and blue color channels

const int IMAGE_FORMAT_RGBA ;
An image with Red, green, blue, and alpha color channels

const int BLEND_NONE ;
Draw a layer or texture as completely opaque.

const int BLEND_ADD ;
Add the color values of a layer or texture to those behind it.

const int BLEND_AVERAGE ;
Average the color values of a layer or texture with those behind it.

const int BLEND_MULTIPLY ;
Mutiply the color values of a lyer or texture with those behind it.

const int LAYER_CULL_BACK ;
Cull the back faces of a layer and render the front.

const int LAYER_CULL_FRONT ;
Cull the front faces of a layer and render the back.

const int LAYER_DRAW_FILL ;
Draw a layer as complete filled-in polygons.

const int LAYER_DRAW_LINES ;
Draw a layer as Lines (a wireframe).

const int LAYER_DRAW_POINTS ;
Draw a layer as a series of points.
